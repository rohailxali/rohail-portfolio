# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-12

### Initial Release
Pixel-perfect Noir portfolio generated for Muhammad Rohail Ali.

#### Features
- ✨ Complete Next.js 14 portfolio with App Router
- 🎨 Custom Noir dark theme (#050505) with orange accent (#FF5A2A)
- 📱 Fully responsive two-column layout (62%-38% split)
- ⚡ Framer Motion animations (parallax, hover effects, page transitions)
- ♿ WCAG AA accessibility compliant
- 📧 Contact form with serverless API route
- 🧪 Jest testing setup with example tests
- 🚀 Vercel deployment configuration
- 🔄 GitHub Actions CI/CD pipeline
- 📝 TypeScript strict mode throughout

#### Components
- Navigation with mobile hamburger menu
- Hero section with large typography and circular headshot
- Services grid with numbered badges (01-04)
- Portfolio masonry grid (3 cols desktop, 2 tablet, 1 mobile)
- Project cards with hover overlays and LQIP
- Project detail modal (Headless UI)
- Contact form modal with validation
- Footer with social links

#### Data
- Personal information (name, contact, links)
- 4 service cards
- 6 example portfolio projects (BRANDING, MOCKUP, LOGO, VIDEO categories)

#### Developer Experience
- ESLint + Prettier configuration
- Example unit tests
- Comprehensive README documentation
- Environment variables template
- Vercel deployment ready
